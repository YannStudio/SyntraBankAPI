# SyntraBankAPI
An unofficial Syntra Bank API 

![Logo](https://imgur.com/C0y2OPo)


- Manage bankaccount:
  - Open a new bankaccount => POST
  - List of transactions => GET
  - List of accounts => GET
  - Make a transaction => POST
  - Deposit money => POST
  - Receive money => POST
  
- Controllers:
  - Transactions
  - Account
  - Client
